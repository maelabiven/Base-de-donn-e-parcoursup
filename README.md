# Base-de-donnee-parcoursup

# Description 
Création d'un script SQL pour une base de donnée sur les admissions avec Parcoursup dans toutes les formations de France. Il fallait creer les tables avec ses attributs ainsi que ses contraintes.

# Objectif
- Maitriser la création de table ainsi que de contraintes
- Comment mettre dans une table différents attributs de différents types

# Outils utilisés
- Visual Studio Code
